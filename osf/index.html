OSF, A Go package support Open Screenplay Format

osf

Experimental golang package for working with Open Screenplay Format 2.0. Open Screenplay Format is an open XML format for screenplays and the native format (when zipped) for Fade In. Two package will include several demonstration command line programs osf2txt which will read a osf file and render plain text in a Fountain like format, txt2osf which takes a plain text file and attempts to render an OSF 2.0 document and finally fadein2txt which will read in a Fade In file and write out plain text in Fountain format.
