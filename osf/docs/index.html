Demonstration programs

When they are writen three demonstration programs are planned.

  • osf2txt - converts Open Screenplay Format 1.2 and 2.0 to plain text (i.e. Fountain)
  • txt2osf - attempts to convert plain text into OSF 2.0 XML
  • fadein2osf - converts Fade In file to Open Screenplay Format
